Formatação em HTML

FORMATAÇÃO HTML

HTML contém vários elementos para definir texto com um significado especial:

Esse texto está em negrito

Esse texto está em itálico

Esse texto está subscrito e sobrescrito.

TIPOS DE ELEMENTOS

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

Exemplo:

<b>-Texto em negrito
<strong>-Texto importante
<i>-Texto em itálico
<em>-Texto enfatizado
<mark>-Texto marcado
<small>-Texto menor
<dell>-Texto excluído
<ins>-Texto inserido
<sub>-Texto subscrito
<sup>-Texto sobrescrito

Agora você sabe usar osa principais elementos de formatação de texto
