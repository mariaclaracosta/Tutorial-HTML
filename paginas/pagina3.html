Atributos HTML

ATRIBUTOS DO HTML

Os atributos do html são informações adicionais para as tags e vem depois da primeira tag.

ATRIBUTO HREF

A <a> tag é um link e o atributo <href> fL para onde o link deverá levar:

Exemplo:

<a href="https://www.youtube.com/"> Esse é um link para o YouTube </a>

E é assim que se usa um href

ATRIBUTO SRC

O catributo <src> especifica o caminho usado para escolher a imagem específica que será exibida.

Exemplo:

<img src="/Tutorial-HTML/imagens/foto.jpeg">

</img>

E é assim que se coloca uma imagem em um site

ATRIBUTO ALTURA E LARGURA

Esses atributos que em inglês são respectivamente height e width também são usados na <img> e definem sua altura e lagura e o tamanho usado é pixels 'px':

Exemplo:

<img height ="150px" width="300px">

</img>

E é assim que se coloca uma imagem em um site
