CABEÇALHOS HTML

Os cabeçalhos são títulos, eles são usados para chamar a atenção e estruturar o site. Para isso é usado o <h> que vai do maior '1' para o menor '6'.

Exemplo:

<h1>

TITULO MAIS IMPORTANTE

</h1>

<h6>
titulo menos importante
</h6>

Agora você já sabe como usar cabeçalhos
