ELEMENTOS DE HTML

TAGS HTML

Elementos são tudo do html, desde a tag inicial até a tag final:

Exemplo:

<p>

Um paragrafo

</p>
<br> Uma quebra de linha
<h3>

Um h3

</h3>

Essas são algumas tags mais usadas

Porém como pode ser visto acima, algumas tags como o <br>, não são fechadas, essa são tags especiais que precisão ser decoradas.
