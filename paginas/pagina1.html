O Basico de HTML

<!DOCTYPE HTML, HTML e BODY>

Todo documento HTML deve começar declarando o seu tipo desse jeito: <!DOCTYPE html>

Depois o HTML gosta de aparecer então você tem que colocar ele de novo e colocar tudo dentro dele, mas dessa vez você deve fechar ele assim: <html></html>

Agora você deve dar corpo ao arquivo literalmente, corpo é body em ingles: <body></body>

Exemplo:

<!DOCTYPE html>

<html>

<body></body>

</html>

Agora você já sabe o básico do básico do básico

CABEÇALHOS

Os cabeçalhos são textos maiores e que chamam mais atenção, eles são representados pela letra 'h' e vão do mais importante, '1' ao menos importante, '6':

Exemplo:

<h1>

CABEÇALHO MAIS IMPORTANTE

</h1>
<h6>
cabeçalho menos importante
</h6>

Agora você já sabe como usar cabeçalhos

PARAGRAFOS

Os paragrafos são paragrafos, eles são usados para escrever os textos de um site e são representados pela letra 'p':

Exemplo:

<p>

Texto aleatório de exemplo

</p>

E é assim que se escreve um paragrafo

LINKS

Os links são representados pela letra 'a' e são formas de ligar algo de outro site ou otro site em si no seu, você coloca o 'a', ' href="" ' e dentro dos dois pontos o link desejado:

Exemplo:

<a href="https://www.youtube.com/"> Esse é um link para o YouTube </a>

E é assim que se usa um link

IMAGENS

As imagens são representadas por 'img' e elas servem para colocar imagens no seu site:

Exemplo:

<img>

</img>

E é assim que se coloca uma imagem em um site
